# US_1024 Afterpay

## User story

As a product owner, \
I want to be able to know how many users would use Afterpay as a payment method.

## Acceptance Criteria

- Afterpay is displayed as an available payment method.
- When selecting Afterpay, a window pops up.
- The window displays a questionary for the user.
- The question displayed allows to pick an answer for the question: \
Would you use Afterpay(yes or no).
