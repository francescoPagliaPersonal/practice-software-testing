# US_1024 Afterpay

## User story

As a user located in the US, \
I want to be able to use Afterpay as a payment method.

## Acceptance Criteria

- Afterpay is displayed as an available payment method.
- When selecting Afterpay, the user is redirected to the Afterpay payment flow.
- If the Afterpay payment fails or is cancelled, the user is returned to the checkout with an appropriate error message.
- The order is not created if the Afterpay payment is unsuccessful.
- The correct order status is set for successful Afterpay transactions.
